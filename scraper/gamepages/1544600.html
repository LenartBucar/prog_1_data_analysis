About This Software

JS Page is built for programming education, project prototyping, auto-tasking and machine learning. Javascript beginners can easily start their learning with JS Page by button clicks within the application. The main goal of this software is giving more options for using Javascript.

Briefly JS Page is a small bridge for using Javascript in a web page and Node. It supports executing Javascript in loaded web page and Node VM by button clicks. And it comes with several features such as node integrated webpage, console window, developer tool, script tag manager, HTML editor, resolution list calculator and color picker. So JS Page is ready for simple front-end web development and testing out your node project with default node modules. And it can be your quick Javascript programming tool.

People can use JS Page like an IDE but JS Page is not an IDE. JS Page insert user's javascript code in a loaded web page. It is designed that way so html file or live web page doesn't get affected by the user's code. So an IDE and JS Page have different purpose. And JS Page has never been designed to replace your IDE. Try JS Page and you will find it is a different tool from your IDE!

Happy programming!
