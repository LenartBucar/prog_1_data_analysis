About This Content

"-Master to Legendary Upgrade is for those who want to purchase only the additional contents if you already have the full game and Master Bundle.
Actual package value when all the individual items in this package are added up equal to $190.

To redeem the items contained in the DLC, go in game and hit the ESC key to bring up the main menu. Click on the 'Community' > ‘Redeem’ menu to begin your item redemption process.

This package contains:
  • Value Pack (75 Days) x1:
    The Value Pack provides the following benefits until it expires:
    Inventory Expansion +16 slots
    Storage Expansion +16 slots
    Combat/Skill/Life/Mount EXP +30%
    Distant Node Investment (Consumes 10 Energy)
    Max Weight Limit +200 LT
    +30% Marketplace Sales upon Silver Collection (Excluding Pearl Items)
    Unlimited Use of Merv's Palette
    Unlimited Beauty Salon
    Value Pack buffs do not stack with each other, but duration does.
    Barter Refresh Points +50
    Required Parley for Barter -10%
    Allows you to remotely check or skip Special Barter

  • Secret Book of Old Moon (15 Days) x1:
    The Secret Book of Old Moon provides the following benefits until it expires:
    Combat EXP +100%
    Skill EXP +50%
    Life EXP +50%
    Reset (enhanced) skills freely
    Change skill add-ons freely
    Change Skill Preset in Safe Zone (Cooldown : 1 hour)
    Buy villa scrolls via your campsite without having a villa invitation
    Stable slot +1 (for all regions)
    Wharf slot +1 (for all regions)

  • Inventory Slot +16 Expansion Coupon x1:
    Use the Inventory Slot Coupon to increase your maximum inventory slots by an additional 16 slots.


  • Premium Outfit Set x1:
    You can obtain a box which contains an outfit suitable for your class.
    Contains a box of class-specific skin.
    Warrior = Goyen Premium Set
    Sorceress = Cartian Premium Set
    Ranger = Sylvia Premium Set
    Berserker = Tantu Premium Set
    Tamer = Daru Premium Set
    Musa = Western Frontiers Premium Set
    Valkyrie = Enslar Premium Set
    Maehwa = Order of the Apricot Premium Set
    Witch = Labreve Premium Set
    Wizard = Lord Red Premium Set
    Ninja = Narusawa Premium Set
    Kunoichi = Ayo Premium Set
    Dark Knight = Sin Terrna Premium Set
    Striker = Blazing Inferno Premium Set
    Mystic = Banha Premium Set
    Lahn = Sanguine Petal Premium Set
    Archer = Luanwulf Premium Set
    Guardian = Ynixtra Premium Set
    Hashashin = Domtur Premium Set
    Nova = Sephia Premium Set


  • Hedgehog (Pet) x1:
    Start your journey with an adorable pet. Hedgehogs have a special skill for life skill EXP.
    Normal Skill: Looting (loots every 4 sec, based on Tier 1 Lv.10)
    Talent: Life EXP Boost (+1%, based on Tier 1)
    The Life EXP Boost Talent increases by +1% for each Tier this pet goes up.
    Special Skill: Gathering Item Quantity Increase

  • Wizard Gosphy (Pet) x1:
    Start your journey with an adorable pet. Wizard Gosphy has a special skill for knowledge.
    Normal Skill: Looting (loots every 4 sec, based on Tier 1 Lv.10)
    Talent: Chance to Gain Knowledge, Chance to Gain Higher-Grade (+5%, based on Tier 1)
    The Chance to Gain Knowledge, Chance to Gain Higher-Grade Talent increases by +1% for each Tier this pet goes up.
    Special Skill: Taunt Monster

  • Contract: [Storage Maid] Fairy Irene x1:
    An employment contract for a maid. You can summon her from anywhere. She can deposit or withdraw one type of item, excluding trade items, from Storage. She can only visit the Storage at the major city of whichever Territory you are currently in.
    Weight Limit: 100 LT
    Cooldown: 20 min

  • Contract: [Transaction Maid] Fairy Irene x1:
    An employment contract for a maid. You can summon her from anywhere. She can move one type of item to and from the Central Market Warehouse.
    Volume Limit: 100 VT
    Cooldown: 20 min

  • Blessing of Kamasylve (15 Days) x1:
    Blessing of Kamasylve provides the following benefits until it expires:
    Energy Recovery +2
    Item Drop Rate +20%
    Central Market Warehouse Capacity +2,000 VT
    Transaction Maid Volume Limit +100 VT
    Storage Transport Weight Limit +1,500 LT

  • Elion's Tear x10:
    Consuming this when your character dies will allow you to resurrect immediately.
    When Combat EXP is decreased from death, you can use it to instantly restore the lost combat EXP.

  • Virtual currency - 2,000 Pearls:
    It's the currency used in the Pearl Shop. It can be purchased with Acoin."
