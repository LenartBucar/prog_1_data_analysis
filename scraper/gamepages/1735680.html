About This Content

This DLC grants access to the digital version of the artbook of Library Of Ruina.

The files are provided in PDF format, and are currently available in Korean and English.

The artbook contains the lore of characters and factions, various illustrations, and chibi sprites. (Beware of possible spoilers!)

After purchasing the Steam DLC for the LOR Artbook, you can access the file through the Library Of Ruina installation path, as described below.

Artbook File Directory: (Installation Path)\artBook\LOR_Artbook_vol1.pdf
Example) E:\Program Files (x86)\SteamLibrary\steamapps\common\Library Of Ruina\artbook\LOR_Artbook_vol1(EN).pdf

You can find the installation path with the Steam client by right-clicking the game's title -> Properties -> Local Files -> Browse...
