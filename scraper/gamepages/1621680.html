About This Game

*** This game is only available in Chinese ***

《仙四》自上市至今，廣受玩家青睞，刻骨銘心的故事軸線，至今仍被各界譽為單機遊戲中的雋永之作。本Steam版本為語音版，遊戲中的主角：雲天河、韓菱紗、慕容紫英及柳夢璃，將活靈活現地為玩家帶來更深刻的感動與真情告白。

迴夢遊仙


山野少年雲天河從小孤獨地成長，整天以追豬、玩豬、獵豬為樂。某日因?盜墓少女韓菱紗而闖入父母的神秘墓室，墓室中的奇事奇景使得山野少年踏上了尋覓往事、求仙問道的旅途。
在尋仙的過程中結識了天仙般美貌的官家千金柳夢璃、擁有劍仙般浩然正氣的慕容紫英。隨著那掩埋在歲月風沙之下的往事日漸清晰，山野少年嘗盡紅塵俗世的喜怒哀樂與悲歡離合，他內心逐漸渴望重新回到往日無憂無慮的日子……

‧ 繾綣柔情

夙世恩怨、曠世情纏，全新演繹淒美動人的愛情故事，讓那些陌生又熟悉的情懷，帶給您心靈深處最難忘的感動！

‧ 瓊樓玉宇

畫面風格脫胎換骨！中原秀美山河、西域市鎮風情、崑崙縹緲仙境，力求再現唐人古風！

‧ 鋒霜劍嘯

獨特的仙術系統，讓您嘯傲五靈！極度個性化的人物特技將配合戰場魄力運鏡。絢麗、刺激的劍術道法各顯神通！

‧ 灼星鑄魂

熔鑄、鍛冶、注靈三大子系統構成的鑄造系統，豐富多變，讓您感受中國古代鑄劍文化的氣氛！

‧ 仙音繞樑

全新配樂，曲數遠超前代！仙音渺渺、柔情似水，首次嘗試古典樂器 LIVE 錄音，更追加一代《蝶戀》 LIVE 重製版回饋玩家！

‧ 陌上繁花

前所未有的個性設計，主角換裝、迷宮路標系統、小地圖標識功能，強調人性化的操作與靈活的互動性！

System Requirements

    Minimum:
    • OS: Windows Vista / Windows 7 / Windows 8 / Windows 10
    • Processor: Intel Core Duo2 3.0Ghz or AMD equivalent (or better)
    • Memory: 512 MB RAM
    • Graphics: Geforce 4 (63 MB or Better)
    • DirectX: Version 9.0c
    • Storage: 3 GB available space
    • Sound Card: Direct compatible sound card for audio
    • Additional Notes: Keyboard and mouse are required
